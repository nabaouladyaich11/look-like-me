# Look Like Me

**Facial Similarity Matching Platform**

Look Like Me is a patent-pending web platform that connects users based on facial similarity using advanced AI. Upload a photo, and the system finds people who look like you.

---

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | Next.js |
| Backend | Django + Django REST Framework |
| Database | PostgreSQL |
| AI Model | AdaFace (ResNet-50) + PyTorch |
| Vector Search | FAISS (Cosine Similarity) |
| Deployment | Docker + AWS |
